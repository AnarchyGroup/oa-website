What is OpenAnarchy?

OpenAnarchy is a cracked anarchy server run on 1.16.5 BungeeCord Spigot software, with a really fast growing peak. Just like other typical anarchy servers, OpenAnarchy has no rules, no staff, and no "kid friendly" chat filters.

How can i join?

OpenAnarchy is playable on Java and Bedrock versions!
You can play on Java with the IP: openanarchy.org Or play on Bedrock with the IP: bedrock.openanarchy.org and with the port 19132.
